Apps Script 설정 방법

  1. Google Drive에서 새 스프레드시트 생성
  2. 상단 확장 프로그램 → Apps Script 클릭
  3. 기존 코드 삭제 후 Code.gs 내용 붙여넣기
  4. 배포 → 새 배포 → 웹 앱 선택
  5. 실행: 나 / 액세스: 모든 사용자(익명 포함)
  6. 생성된 URL을 위에 입력 후 저장
