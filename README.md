# 📚 고3 학습 플래너

2026년 5월 16일 ~ 11월 19일 수능 D-day 학습 관리 웹앱

## 📁 파일 구성

| 파일 | 설명 |
|------|------|
| `index.html` | 플래너 웹앱 (이것만 GitHub에 올리면 됨) |
| `Code.gs` | Google Apps Script 백엔드 (구글 시트 연동) |

---

## 🚀 배포 순서

### 1단계 — GitHub Pages 설정 (URL 발급)

1. **github.com** 로그인 후 우측 상단 `+` → **New repository**
2. Repository name: `planner` (또는 원하는 이름)
3. **Public** 선택 → **Create repository**
4. `index.html` 파일을 드래그하거나 **Add file → Upload files**로 업로드
5. **Settings → Pages → Branch: main, / (root) → Save**
6. 약 1~2분 후 `https://[내아이디].github.io/planner/` URL 생성

> 이 URL을 브라우저에서 열거나 스마트폰 홈 화면에 추가하면 앱처럼 사용 가능

---

### 2단계 — Google Apps Script 설정 (시트 연동)

1. **[sheets.new](https://sheets.new)** 접속 → 새 스프레드시트 생성
   - 제목: `고3 학습 플래너` (자유)
2. 상단 메뉴 **확장 프로그램 → Apps Script** 클릭
3. 기존 코드 전체 삭제 후 `Code.gs` 내용 붙여넣기
4. 💾 **저장** (Ctrl+S)
5. 상단 **배포 → 새 배포** 클릭
6. 설정:
   - 유형: **웹 앱**
   - 설명: `학습플래너`
   - 실행 계정: **나**
   - 액세스 권한: **모든 사용자(익명 포함)**
7. **배포** 클릭 → 권한 허용 → **웹 앱 URL 복사**

---

### 3단계 — 플래너에서 URL 연결

1. 발급된 GitHub Pages URL을 스마트폰/PC에서 열기
2. 하단 **설정** 탭 클릭
3. 복사한 Apps Script URL 붙여넣기
4. **저장 & 연결 테스트** → ✅ 연결 성공 확인

---

## 📱 스마트폰 홈 화면 추가 방법

### iPhone (Safari)
1. Safari에서 플래너 URL 열기
2. 하단 공유 버튼(□↑) 탭
3. **홈 화면에 추가** 선택

### Android (Chrome)
1. Chrome에서 플래너 URL 열기
2. 주소창 우측 메뉴(⋮) 탭
3. **홈 화면에 추가** 선택

## ✅ 기능 요약

| 탭 | 기능 |
|----|------|
| 📚 플래너 | 주간 이동, 일별 과목 체크, 메모, 구글 시트 저장 |
| 📅 달력 | 월간 달성 현황, 날짜 클릭 시 해당 날로 이동 |
| 📊 통계 | 전체/월별/과목별 달성률, 연속 미완료 문제 과목 경고 |
| ⚙️ 설정 | GAS URL 관리, 연결 테스트, 로컬 캐시 초기화 |

### 학습 목표
- **월~금**: 수학(6시간), 국어(3시간), 영어단어(1시간), 사탐(하교 후 1시간)

## ❓ 자주 묻는 질문

**Q. Apps Script 재배포 해야 하나요?**  
A. Code.gs 수정 시에만 재배포 필요. `배포 → 배포 관리 → ✏️ 수정`으로 기존 URL 유지 가능.

**Q. 여러 기기에서 동시 사용 가능한가요?**  
A. 가능합니다. 구글 시트에 저장 후 다른 기기에서 열면 데이터가 불러와집니다.

**Q. 구글 시트는 어디서 확인하나요?**  
A. Google Drive에서 생성한 스프레드시트를 열면 `학습기록` 시트에서 확인 가능.
